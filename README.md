## ЁЯй╕ What is KURUMI?

**KURUMI** is a fully offline, privacy-first desktop application that brings the full power of large language models to your local machine тАФ zero subscriptions, zero data leaks, zero cloud dependency. Run frontier-class AI on your own hardware. Own your data completely.

Inspired by the visual brutality of **Jujutsu Kaisen**, KURUMI's "Cursed Blood" interface bleeds deep crimson through neo-glassmorphism panels, glowing vein-like borders, and particle energy effects. It doesn't just run AI тАФ it *channels* it.

## тЬж Current Features

### ЁЯТм Chat Interface
- **Real-time streaming** тАФ tokens appear as they generate, with an animated "Summoning from the void..." loading state
- **Conversation sidebar** тАФ full chat history with search, pin/unpin, and delete
- **Markdown rendering** тАФ rich formatted output with syntax-highlighted code blocks (Cursed Blood dark theme), tables, lists, blockquotes, and more
- **Copy button** on every code block тАФ one click to clipboard
- **System prompt** тАФ every conversation is pre-seeded with formatting instructions so the model uses Markdown automatically
- **Auto-scroll** тАФ chat window follows the stream in real time
- **Multi-turn memory** тАФ full conversation history sent to the model on every message

### ЁЯЦ╝я╕П Image Generation Studio
- **Automatic1111** тАФ txt2img and img2img with sampler / steps / CFG / size / seed, optional checkpoint override, denoise control for img2img
- **Save to disk** тАФ one-click export of the current preview into the app data directory as PNG
- **ComfyUI** тАФ quick connection test to a local Comfy server (full workflow queue is not wired in this build)

### ЁЯза Model Management
- **Installed models page** тАФ see all local Ollama models with size, parameters, quantization level, and family
- **Image checkpoints (A1111)** тАФ separate panel to list Stable Diffusion checkpoints from your WebUI API and mark which one Image Gen should use
- **One-click select** тАФ switch active model from the Models page
- **Pull new models** тАФ download directly from inside the app with a real-time streaming progress bar (%)
- **Delete models** тАФ with double-confirm safety guard

### ЁЯЫНя╕П Model Store
- **Dual-source browser** тАФ browse live from **Ollama Library** and **HuggingFace Hub** simultaneously
- **Catalog scope** тАФ filter search results toward **language / chat models** vs **image & diffusion** (plus *All*) on both tabs
- **HuggingFace GGUF search** тАФ sorted by Most Downloaded / Liked / Newest
- **Quantization picker** тАФ see all available `.gguf` variants per HuggingFace model with file sizes
- **Direct install** тАФ `ollama pull hf.co/org/repo:Q4_K_M` wired directly to streaming progress modal
- **Pagination** тАФ navigate pages of results
- **Installed detection** тАФ already-installed models show a green "Installed" badge across both sources

### ЁЯОи Aesthetics ("Cursed Blood" Theme)
- **Deep void background** `#050305` with floating red particle emitters
- **Glassmorphism panels** тАФ `backdrop-filter: blur(16px)` layered glass throughout
- **Red accent system** тАФ `#8B0000` тЖТ `#C41E3A` тЖТ `#FF2244` gradient hierarchy
- **Glowing borders** тАФ red vein-like borders with radial `box-shadow` on active elements
- **Animated loading states** тАФ pulsing orb, streaming cursor, gradient progress bars
- **Frameless window** тАФ custom title bar with minimize/maximize/close controls

### ЁЯЧДя╕П Data Persistence
- **SQLite database** тАФ all conversations and messages stored locally via `better-sqlite3`
- **Full-text search** тАФ FTS5 index on message content
- **Conversation hydration** тАФ last conversation automatically restored on app restart

## тЬж Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/bhoomik-codes/kurumi.git
cd kurumi/kurumi
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start Ollama

```bash
# Start Ollama in the background (if not already running)
ollama serve

# Pull a model to get started (choose one)
ollama pull llama3.2:3b      # Recommended starter тАФ fast, 2 GB
ollama pull qwen2.5:7b       # Better reasoning, 4.4 GB
ollama pull deepseek-r1:7b   # Best reasoning, 4.7 GB
```

### 4. Launch in development mode

```bash
npm run dev
```

The Electron window will open automatically.
